2024-2025 Ukrainian Youth Academy of Science competition project. 1-st place Regionaly & 4-th place Nationally.
Kohonens Neural Network using Learning VectorQuantization built by scratch in Python is used to clasterise and later predict trends of the timelines. BTC prices are used as an example.
The software is built exclusively by hand, without external libraries for the educational purposes. Proving it's effectiveness with 2-5% relative error in predicting BTC price trend for the week.
